⬇ Exportar datos

Descargue todos los datos actuales del sistema en formato JSON. El archivo incluye empresas, visitas, hallazgos, oportunidades y contactos. Puede usarlo como respaldo o para compartir con otro usuario.

⬆ Importar datos

Cargue un archivo JSON exportado desde VisitaLab. Elija cómo combinar los datos:

➕
Agregar como nuevos
Los datos importados se suman a los existentes. No se elimina nada. Los IDs se reasignan para evitar conflictos.
🔄
Sobreescribir todo
Reemplaza completamente los datos actuales con los del archivo. Los datos existentes se perderán.

🗑 Zona de peligro — Vaciar datos

Elimine de forma permanente grupos de datos del sistema. Estas acciones no se pueden deshacer. Se recomienda exportar un respaldo antes de proceder.

📋 Visitas
Elimina todas las visitas registradas y sus hallazgos, contactos y oportunidades asociadas.
🏢 Empresas
Elimina todas las empresas. Solo se puede hacer si no hay visitas asociadas.
🔍 Hallazgos
Elimina todos los hallazgos y sus oportunidades relacionadas, conservando las visitas.
◈ Oportunidades
Elimina todas las oportunidades de mejora, conservando visitas y hallazgos.
⚠ Todo el sistema
Elimina absolutamente todos los datos. La sesión continuará abierta pero sin información.
